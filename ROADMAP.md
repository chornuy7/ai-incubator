## О проекте

Панель автоматизации и парсинга Telegram в тёмном дизайне ai-incubator.
Портированы фичи парсинга из GRAMGPT + TGStat-парсер, плюс модули «инкубатора»
(комментинг/чаттинг/реакции/прогрев и т.д.).

**Стек:** React 18 + TypeScript + Vite 5 + Tailwind 3 (фронт) · Node.js Express (ESM) +
GramJS (MTProto) + cheerio (бэк) · zustand, react-router v6, lucide-react.
Прогресс задач — polling (без websockets). Данные модулей — JSON-файлы в `server/data/`.

**Запуск:** `npm install` → `npm run dev` (web :5173 + api :3001).
Секреты (`.env`, cookies, `.session`, `server/data/`) — в `.gitignore`, в репо их нет.

## ✅ Что работает (по модулям)

Все 12 модулей имеют **реальный бэкенд-воркер** (`server/modules/workers.js`, карта `WORKERS`),
а не заглушку — используют живые вызовы GramJS.

### Парсеры (наши, протестированы на 5 живых аккаунтах)

| Модуль | Ключ | Статус | Что делает |
|---|---|---|---|
| Парсинг каналов | `parsing` | ✅ live | Поиск каналов по ключам/окончаниям, фильтры участников, комментов, дедуп, ★-рейтинг, экспорт |
| Парсер групп | `parsing-groups` | ✅ live | То же для групп, шаблоны, ИИ-подсказки ключей |
| Парсер пользователей | `parsing-users` | ✅ live | Участники чатов + режим «кто писал» (роли: админ/премиум/юзер), **пересечение аудиторий** |
| Парсер по сообщениям | `parsing-messages` | ✅ live | Юзеры по ключам в сообщениях, фильтр по датам, статистика активности |
| Парсер комментариев | `parsing-comments` | ✅ live | Юзеры из комментов к постам, фильтры, лимиты постов/комментов |
| TGStat-парсер | `tgstat` | ✅ live | Каталог по категориям/регионам + **расширенный поиск** с фильтрами (по cookies) |

Проверено вживую: каналы ~22, группы ~10, users ~12, writers ~134, messages ~87,
comments ~149, TGStat search 30–60 результатов с фильтрами.

### Модули инкубатора (бэкенд реальный; системный live-прогон на 5 акках — в бэклоге)

| Модуль | Ключ | Бэкенд | Реальные действия в TG |
|---|---|---|---|
| Нейрокомментинг | `neuro-commenting` | ✅ | join + `sendChannelComment` + генерация OpenAI |
| Нейрочаттинг | `neuro-chatting` | ✅ | `sendMessage` в группах, контекст диалога |

### Инфраструктура (работает)

- **Менеджер аккаунтов**: добавление, фильтры по странам/ролям, прокси.
- **Локи аккаунтов**: 1 аккаунт = 1 задача глобально (защита от параллельного использования).
- **ИИ-защита**: 3 уровня (Консервативный / Сбалансированный / Агрессивный).
- **Задержки** между действиями/чатами + **обработка FloodWait** с карантином.
- **Папки целей**: сохранение результатов парсинга в БД, догрузка в существующую папку с дедупом,
  **валидация** (`POST /target-folders/:id/validate` — резолвит каждую цель, удаляет мёртвые).
- **Экспорт**: CSV + Excel (`.xls`, SpreadsheetML).
- **История парсинга**: агрегирует задачи всех модулей + TGStat-импорты, просмотр + экспорт.
- **Help Center**: ответы через OpenAI (`gpt-4o-mini`), при отсутствии ключа — шаблонные ответы.
- **Чёрный список** целей, авто-добавление при ошибках.
